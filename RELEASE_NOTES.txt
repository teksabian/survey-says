CHANGES IN THIS RELEASE
=============================================================================

🎨 UI/UX IMPROVEMENTS (8 fixes)
-------------------------------

1. HOST LOGIN PAGE REDESIGN
   - ✅ Removed instruction box (no longer shows default PIN)
   - ✅ Removed lock icon 🔒
   - ✅ Increased logo size (150px → 250px)
   - ✅ Added "Pub Family Feud" title above logo
   - Result: Cleaner, more professional login screen

2. SETTINGS PAGE REORGANIZATION
   - ✅ Added "← Return to Dashboard" button (top left)
   - ✅ Kept scroll layout (no tabs - simpler)
   - ✅ Reordered sections for logical flow:
     * 🎮 Game Controls
     * 📢 Broadcast Message
     * 💤 Put Server to Sleep (NEW!)
     * 🔄 Reset Game (moved from bottom)
     * 🎯 Game Configuration
   
3. BROADCAST MESSAGE DISMISSAL
   - ✅ Added X button to close broadcast popup (player side)
   - Players can now dismiss messages without host clearing them
   - Message stays dismissed until new broadcast or page refresh

4. TIEBREAKER TEXT UPDATE (Mobile)
   - Changed from: "How many people surveyed by Family Feud gave the #1 answer?"
   - Changed to: "How many people out of the 100 surveyed by Family Feud gave the #1 answer? (You do not have to have answer on your sheet)"
   - Clearer instructions for pub environment

5. CONNECTION STATUS INDICATOR (Mobile)
   - ✅ Green dot on answer submission page
   - ✅ Positioned on same line as "Round # | Team Name" (right side)
   - ✅ Pulses every 5 seconds
   - ✅ Green = Connected | Grey = Disconnected
   - Provides visual feedback that page is live/working

6. VERSION NUMBER UPDATE
   - Changed to v1.0.0 for tracking
   - After V2→V1 merge: Will change to v2.0.0
   - Helps identify which codebase at a glance

7. PER-TEAM "VIEW ANSWER" BUTTON
   - ✅ Added "👁️ View Answer" button on scoring queue
   - ✅ Positioned to left of team code field
   - ✅ Opens popup showing ONLY that team's answers
   - Different from "SHOW ALL ANSWERS" button
   - Faster workflow for reviewing individual submissions

8. REORGANIZED SCORING QUEUE BUTTONS
   - ✅ Moved "SHOW ANSWERS" from floating position to menu bar
   - New layout: [Back] [Refresh] [View Scored Teams] [SHOW ANSWERS]
   - Cleaner, more organized interface

=============================================================================

🛡️ CRITICAL PROTECTIONS (4 fixes)
----------------------------------

9. CONFIRMATION DIALOGS
   - ✅ "Are you sure?" popup for:
     * Reset Game button
     * Delete All Rounds
     * Deactivate Active Round
   - Prevents catastrophic accidental clicks during game
   - **HIGH PRIORITY** - Protects against disasters

10. TEAM NAME CHARACTER LIMIT
    - ✅ Limited to 30 characters maximum
    - Prevents UI breaking with long names
    - Clean truncation with "..." if needed
    - Validation on both client and server side

11. DUPLICATE TEAM NAME PREVENTION
    - ✅ Checks if team name already exists
    - ✅ Suggests alternatives: "Winners 2", "Winners 3", etc.
    - Prevents confusion with multiple teams having same name
    - Clear error message with suggested fix

12. MOBILE ANSWER VALIDATION
    - ✅ Warns if player submits blank answers
    - Popup: "You have blank answers - submit anyway?"
    - Prevents accidental empty submissions
    - Player can confirm if intentional
