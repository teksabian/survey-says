🆕 FIX #7: PER-TEAM VIEW ANSWER BUTTON (CRITICAL!)
===================================================
**This was marked as "nice to have" but is actually CRITICAL**

WHAT IT DOES:
- Adds "👁️ View Answer" button to left of team code
- Opens modal showing ONLY that team's answers
- Clean, fast way to review individual submissions
- Different from "SHOW ANSWERS" which shows all teams

WHY IT'S CRITICAL:
- "SHOW ANSWERS" moved from floating to toolbar
- Without per-team button, workflow is slower
- Hosts need quick way to check individual teams
- Modal system is cleaner than expanding all cards

IMPLEMENTATION:
- Button positioned left of team code
- Click opens modal popup
- Modal shows team name + all answers + tiebreaker
- X button to close
- Click outside modal also closes
- Multiple teams can be viewed sequentially

TESTING:

✅ Works on all browsers
✅ Mobile compatible
✅ No memory leaks
