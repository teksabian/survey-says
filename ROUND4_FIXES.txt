Fix #5: Score Edit History/Undo ✅
-----------------------------------
**Problem:**
- Can't see previous score when editing
- Easy to accidentally overwrite
- No undo if you mess up

**Solution:**
- **Previous score stored before each edit**
- Database migration adds `previous_score` column
- Edit page shows:
  - Current Score: 24 points
  - Previous Score: 18 points (+6)
- **Revert button** to undo last change
- One-level undo (last change only)

**Example:**
```
Edit Score for Team "Winners"

Code: ABCD • Current Score: 24 points • Tiebreaker: 45
📊 Previous Score: 18 points (+6)

[Cancel] [↩️ Revert to 18] [💾 Save Changes]
```

**How It Works:**
1. Team scored at 18 points
2. Edit score to 24 points → previous_score = 18
3. Edit again to 30 points → previous_score = 24
4. Click "Revert to 24" → score = 24, previous_score = 30
5. Can revert back and forth

**Files Modified:**
- app.py:
  - Added previous_score migration (line 193-199)
  - update_score function stores previous (line 1388)
  - New revert_score route (line 1397-1410)
- templates/edit_score.html:
  - Shows previous score with diff
  - Revert button added
