## 🔍 **IF SOMETHING FAILS**

### **Check Logs First!**
```bash
# View latest log
cat logs/$(ls -t logs/ | head -1)

# Search for errors
grep ERROR logs/*.log
```

### **Common Issues:**

**Logo not showing:**
```bash
# Verify files exist
ls static/logo.png static/favicon.png

# Hard refresh browser: Ctrl+F5
```

**Auto-check not working:**
- Make sure answers match (case-insensitive)
- Check log for "auto_checks" debug info
- Verify you're using V2 code (check app.py has logging)

**Round summary crashes:**
- Check logs for stack trace
- Verify tiebreaker values in database
- Both teams should have tiebreaker or None (both OK now)
