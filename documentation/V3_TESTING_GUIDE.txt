## 📞 **TROUBLESHOOTING**

**Issue: Auto-refresh not working**
- Check browser console for errors
- Verify API endpoint: `curl http://YOUR_IP:5000/host/codes-status`
- Check logs: `cat logs/*.log | grep "Codes status"`

**Issue: Flash not showing**
- Hard refresh browser: Ctrl+F5
- Check CSS loaded (inspect element)
- Verify flash animation exists in styles

**Issue: Spinner not appearing**
- Wait full 10 seconds
- Check if spinner element exists (inspect)
- Verify refresh happens (network tab)

**Issue: Logo not showing**
- Verify: `ls static/logo.png`
- Hard refresh: Ctrl+F5
- Check path in HTML
