## 🎨 UI IMPROVEMENTS (4 Changes)

### **1. Cleaner Host Dashboard**
**Removed:** "No submissions yet - waiting for teams..." message  
**Reason:** Unnecessary UI clutter  
**File:** `templates/host.html`

### **2. Auto-Refresh Team Play Page**
**Added:** JavaScript polling (every 5 seconds) to detect:
- Round closes → Auto-reload to "waiting for next round" page
- New round activates → Auto-reload to show new submission form
- Round changes → Immediate reload

**Problem Solved:** Teams no longer stuck on old forms when:
- Host closes round while team is filling out form
- Host activates new round while team is waiting
- Team doesn't submit and leaves phone idle

**File:** `templates/play.html`

### **3. Skip Scoring Queue When All Scored**
**Current behavior:** Click "END ROUND & SCORE" → Empty scoring queue → Click "Announce Winner"  
**New behavior:** Click "END ROUND & SCORE" → Directly to winner announcement (if all scored)

**Logic:** If round is closed AND no unscored submissions exist, skip empty queue page

**File:** `app.py` - `close_round()` function

### **4. Auto-Fill Team Name in Manual Entry**
**Added:** JavaScript that auto-fills team name when code selected  
**Behavior:** 
- Select "AMYC - test" from dropdown
- Team name field auto-fills with "test"
- Field remains editable (host can change if needed)

**File:** `templates/manual_entry.html`
