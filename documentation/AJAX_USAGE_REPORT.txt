## 🔴 **SHOULD USE AJAX (Currently Page Reloads)**

### **PRIORITY #1 - Scoring Team Submissions** 🚨 **HIGH PRIORITY**
**Location:** scoring_queue.html  
**Current:** Form POST → Page reload → Answers hidden  
**Problem:**
```
Host flow:
1. Click "Show Answers"
2. Score Team 1 → Submit → PAGE RELOAD
3. Click "Show Answers" AGAIN
4. Score Team 2 → Submit → PAGE RELOAD
5. Click "Show Answers" AGAIN
...repeat for every team!
```

**Should Be:**
```javascript
// AJAX submission
async function submitScore(submissionId) {
    const formData = new FormData(form);
    const response = await fetch('/host/score-team/' + submissionId, {
        method: 'POST',
        body: formData
    });
    
    if (response.ok) {
        // Remove team card from page
        document.getElementById('team-' + submissionId).remove();
        // Show toast
        showToast('Team scored!', 'success');
        // Update counter
        updateUnscoredCount();
    }
}
```

**Benefits:**
- ✅ No page reload
- ✅ Answers stay visible
- ✅ Team card just disappears
- ✅ Smooth workflow
- ✅ Fast scoring

**Impact:** **MASSIVE** - Fixes annoying workflow bug!

## 💡 **KEY INSIGHTS**

**Good AJAX Use Cases:**
- ✅ Live polling/updates (codes, counts, status)
- ✅ Actions that remove items from a list
- ✅ Repeated workflow actions (scoring multiple teams)
- ✅ Stay-on-page operations (manual entry)

**Bad AJAX Use Cases:**
- ❌ Authentication/login (need redirect)
- ❌ Page navigation (join → play)
- ❌ Major destructive operations (reset)
- ❌ File uploads (unless progressive upload needed)
- ❌ When user expects page change
