## 🎮 DASHBOARD FEATURES

- Upload your answer sheet DOCX

When round is active and has answers:

## 📋 COMPLETE WORKFLOW

### **Setup (One Time):**
1. Start server
2. 25 codes auto-generated
3. Print code sheet
4. Hand to tables

### **Choose Creation Method:**

**Option A - Manual:**
1. Click "Create Round Manually"
2. Fill in ALL 8 rounds
3. Submit → All rounds created, Round 1 active

**Option B - DOCX:**
1. Click "Upload Answer Sheet"
2. Select DOCX file
3. Upload → All 8 rounds created, Round 1 active

### **Game Play (8 Rounds):**

**For Each Round:**
1. Round is active (answers locked with 🔧 icons)
2. Teams submit on phones (or paper via manual entry)
3. Submission count updates every 10 seconds
4. Click "Score Submissions"
5. Manually check correct answers
6. Submit each team individually
7. All teams scored → "View Round Summary"
8. See winner + tiebreaker details
9. Announce to pub
10. Click "Start Round 2"
11. Repeat for rounds 2-8

### **Paper Submissions:**
1. Team gives you paper answers
2. Write code on their sheet (e.g., H4D7)
3. Click "Manual Entry (Paper Submissions)"
4. Select their code from dropdown
5. Enter team name and answers
6. Submit → Goes to scoring queue
7. Score same as phone submissions
