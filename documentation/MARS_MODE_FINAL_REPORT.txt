## 🔴 **CRITICAL ISSUES (3)**

### 1. **XSS in Broadcast Messages** 🚨
**Risk:** HIGH  
**Exploitability:** Easy (malicious host)  
**Impact:** JavaScript runs on all team phones

**What Happens:**
```
Host types: <script>alert('Hacked!')</script>
System stores it → Displays on phones → Code runs!
```

**Fix:** HTML escape on input (2 minutes)

---

### 2. **Orphaned Submissions Crash Leaderboard** 🚨
**Risk:** MEDIUM  
**Exploitability:** Hard (requires DB access)  
**Impact:** Game stops working

**What Happens:**
```
Team submits → Admin deletes team → Leaderboard shows NULL → Crash!
```

**Fix:** NULL check in templates (2 minutes)

---

### 3. **Missing Answer Column Safety** 🚨
**Risk:** MEDIUM  
**Exploitability:** Hard (database corruption)  
**Impact:** Scoring fails

**What Happens:**
```
Round has 6 answers → Submission missing answer4-6 → NULL errors
```

**Fix:** Use .get() with NULL checks (3 minutes)
