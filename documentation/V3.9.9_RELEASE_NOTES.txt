## 🐛 BUGS FIXED (7 Total)

### **CRITICAL FIXES (3)**

**1. Multiple Active Rounds**
- System allowed 3-7+ rounds active simultaneously
- Teams saw different rounds, total chaos
- **FIX:** Added `BEGIN IMMEDIATE` transaction + deactivate all before activating new one

**2. Race Condition on Activation**
- Two hosts could activate different rounds at once
- Both became active
- **FIX:** Database locking prevents concurrent activation

**3. Out-of-Order Activation**
- Activating rounds out of sequence compounded bug #1
- **FIX:** Same transaction fix handles this

---

### **HIGH PRIORITY FIXES (2)**

**4. Activate Round Without Answers**
- Host could activate round before setting answers
- Game broke, teams couldn't be scored
- **FIX:** Validation checks if answer1 exists before allowing activation

**5. Duplicate Submissions**
- WiFi drops allowed teams to submit twice
- **FIX:** Explicit duplicate check before insert + friendly message

---

### **WORKFLOW FIXES (2)**

**6. QR Code Auto-Fill Removed**
- QR codes were auto-filling team codes (wrong!)
- **FIX:** QR now points to `/join` only, team enters code manually

**7. QR Code Port Number**
- Default URL missing :5000 port (teams couldn't connect!)
- **FIX:** Default is now `http://gamenightguild.net:5000`

## ✅ INSTALLATION

### **FIRST TIME SETUP:**

1. **Unzip the package**
2. **Run the server:**
   ```
   Double-click: RUN_SERVER.bat
   OR
   Command line: python app.py
   ```
3. **Configure QR Base URL:**
   - Go to `http://localhost:5000/host` (PIN: 6551)
   - Click Settings
   - Update QR Base URL to your local IP + port
   - Example: `http://192.168.1.100:5000`
   - Save settings
4. **Generate team codes and play!**

---

### **UPGRADE FROM V3.9.7/V3.9.8:**

1. **Backup your database:**
   ```
   Copy feud.db to feud_backup.db
   ```
2. **Stop server** (Ctrl+C)
3. **Replace app.py** (from this package)
4. **Replace templates/ folder** (from this package)
5. **Restart server**
6. **Update QR Base URL in Settings** (add :5000 if missing)

**Database is fully compatible - no migration needed!**

## 🚨 TROUBLESHOOTING

**"Internal Server Error"**
→ Make sure templates/ folder exists

**"QR code doesn't work"**
→ Check QR Base URL includes :5000 port  
→ Example: `http://192.168.1.100:5000`

**"Can't activate round"**
→ Set answers for the round first

**"Teams can't connect"**
→ Verify firewall allows port 5000  
→ Verify teams on same WiFi network

**"Database locked"**
→ Normal with concurrent operations, wait 1s and retry
