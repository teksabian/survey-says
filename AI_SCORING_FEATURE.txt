🤖 AI-ASSISTED SCORING SYSTEM
==============================

WHAT IT DOES:
-------------
Host can optionally send any team's answers to Claude AI for automated scoring.
AI evaluates semantic matches between team answers and survey answers.
Host reviews AI suggestions and can accept, modify, or ignore them.

HOW IT WORKS:
-------------

1. Host goes to Scoring Queue
2. Sees a team's submission
3. Clicks "🤖 Send to AI for Scoring" button
4. AI analyzes:
   - The question
   - Survey answers (with point values)
   - Team's submitted answers
   
5. AI returns:
   - Which team answers match which survey answers
   - Confidence level for each match
   - Semantic matching (not exact text)
   
6. UI automatically:
   - Unchecks all boxes
   - Checks the boxes AI suggests
   - Updates score display
   
7. Host can:
   - Accept AI suggestions (just click Submit)
   - Modify (uncheck/check different boxes)
   - Ignore completely (manual score as before)

EXAMPLES OF WHAT AI UNDERSTANDS:
---------------------------------

Question: "Name something you drive"

Survey Answer #1: "automobile" (45 pts)
Team Answer: "car"
AI: ✅ MATCH - semantically the same

Survey Answer #6: "bicycle" (2 pts)
Team Answer: "bike"
AI: ✅ MATCH - common abbreviation

Survey Answer #4: "van" (10 pts)
Team Answer: "minivan"
AI: ✅ MATCH - specific type of van

Team Answer: "boat"
No survey answer about boats
AI: ❌ NO MATCH

TECHNICAL DETAILS:
------------------

Backend:
- Uses Claude Sonnet 4 API
- Endpoint: POST /host/ai-score/<submission_id>
- Function: score_with_ai() in app.py

Frontend:
- Button: "🤖 Send to AI for Scoring"
- JavaScript: scoreWithAI() function
- Auto-updates checkboxes based on AI response

API Requirements:
- ANTHROPIC_API_KEY environment variable
- anthropic==0.18.1 Python package
- Model: claude-sonnet-4-20250514

COST:
-----
~$0.01 per scoring (1 penny)
100 scorings = ~$1
Extremely cheap for testing and production use

SETUP INSTRUCTIONS:
-------------------

1. Get Anthropic API Key:
   - Go to https://console.anthropic.com/
   - Sign up / Log in
   - Create API key
   - Copy it (sk-ant-api03-...)

2. Add to Render:
   - Go to Render dashboard
   - Find v2 service
   - Environment → Add Variable
   - Key: ANTHROPIC_API_KEY
   - Value: sk-ant-api03-...
   - Save

3. Deploy:
   - git push origin v2-development
   - Render auto-deploys
   - Feature is live!

FALLBACK BEHAVIOR:
------------------

- Clicking shows error: "AI scoring not configured"
- Manual scoring still works perfectly

If API call fails:

- Button resets
